4. Install

Mac Install Help

  1. Download the correct Mac zip and unzip it.
  2. Drag Subtitle Converter.app into Applications.
  3. Try opening it once.
  4. If macOS blocks it, go to System Settings > Privacy & Security and click Open Anyway.
  5. If needed, Control-click the app in Applications and choose Open.

These builds are unsigned, so some Macs will show a warning the first time. Apple’s current guidance for allowing an app is in Open a Mac app from an unknown developer.

Windows: unzip the download, then open Subtitle Converter.exe.
